Platform

- NUCLEO-F030R8 evaluation board based on stm32f030r8t6 MCU
  -- http://www.st.com/en/evaluation-tools/nucleo-f030r8.html

- Wireless Gateway Shield v1.0
  -- http://wiki.seeed.cc/WireLess_Gate_Shield_V1.0

Pin usage:
- SWD: PA13, PA14
- UART TX: PB6
- User LED: PA5
- SPI for nRF24L01 (wirelss gateway shield)
  -- SPI1: PA5, PA6, PA7
  -- CSN: PA9
  -- CE: PA8
